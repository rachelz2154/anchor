# anchor
Anchor is an ambient agent that detects unconscious task switching during a declared focus window and inserts a real-time decision checkpoint.
When drift is detected, it asks one question — “On purpose?” — requiring the user to consciously choose whether to continue or switch tasks.
Unlike traditional habit trackers, Anchor intervenes at the moment of drift — where behavior actually changes.
Designed for smart glasses (Even G2) to leverage real-world context signals.
